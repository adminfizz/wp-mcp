# เป้าหมายโปรเจกต์ + สิ่งที่เจ้าของสั่งทั้งหมด (บันทึกไว้อ้างอิง)

> ไฟล์นี้สรุปความต้องการของเจ้าของจากการคุยทั้งหมด ใช้เป็น "ธงเป้าหมาย" ของโปรเจกต์

---

## 🎯 เป้าหมายหลัก (ใจความเดียว)
มี **ปลั๊กอิน SEO** อยู่บน WordPress แล้ว ปกติต้อง **login เข้า wp-admin ไปกดใช้งานเอง**
ต้องการ **สั่งกดผ่าน Telegram แทน** ให้มันรันอัตโนมัติ — แยกได้หลายโดเมน — **จบ**

---

## 🧠 แนวคิด (รีโมทคอนโทรล)
```
ปลั๊กอิน SEO เดิม  =  เครื่อง (ตัวทำงานจริง — ไม่แก้โค้ดมัน)
ระบบที่สร้าง       =  รีโมท + ตัวรับสัญญาณ (Telegram bot + MCP + bridge/connector)
Telegram          =  ปุ่มกด (แทนการ login wp-admin)
```
ไม่ได้แก้ปลั๊กอินเดิม — แค่ "เสียบสายรีโมท" เข้าไปเรียก/กดมันแทน

## 🔌 วิธีเชื่อมปลั๊กอิน SEO เดิม (2 ทาง — เลือกตามที่ปลั๊กอินเปิดทาง)
1. **ปลั๊กอินมี REST API** → `/setworkflow <เว็บ> <json ของ API>` ครั้งเดียว → สั่งด้วย `/topic <เว็บ> <หัวข้อ>` / ดูสถานะ `/jobs <เว็บ>`
2. **ปลั๊กอินมีแค่ปุ่มใน wp-admin** → ใส่ snippet 3 บรรทัด (เรียกฟังก์ชันเดียวกับที่ปุ่มกด) → สั่งด้วย `/action <เว็บ> <ชื่อ>`

ดูรายละเอียด [CONNECT-EXISTING.md](./CONNECT-EXISTING.md)

## ⏳ เหลือจุดเดียวที่ทำให้ "จบ"
ต้องรู้ว่าปลั๊กอิน SEO เดิม **trigger ยังไง**:
- **ชื่อปลั๊กอิน SEO** (เพื่อหาว่ามี REST API ไหม)
- ตอนใช้ **กดตรงไหนใน wp-admin** (เมนู → ปุ่ม)

เมื่อได้ข้อมูลนี้ → เสียบ `/setworkflow` หรือ snippet `/action` → **สั่งปลั๊กอิน SEO ผ่าน Telegram ได้จริง = จบ**

## ✅ แนวทาง "สร้างโครงก่อน เสียบปลั๊กอินทีหลัง" — ถูกต้อง
ตอนนี้ยังไม่มีไฟล์ปลั๊กอิน SEO อยู่ในมือ — **ไม่เป็นปัญหา** เพราะระบบออกแบบมาแบบ **plugin-agnostic** (โครงไม่ผูกกับปลั๊กอินตัวใดตัวหนึ่ง) จึงทำโครงไว้ก่อนได้ แล้วค่อยเสียบทีหลัง

**สำคัญ:** "เอาปลั๊กอินมารวม" = เชื่อม**ตอนรันจริงผ่านเน็ต** ไม่ใช่เอาไฟล์ปลั๊กอินมาใส่ในรีโปนี้
- ปลั๊กอิน SEO รันอยู่บนเว็บ WordPress ของมันเอง
- ระบบเราเชื่อมเข้าหา: ถ้ามี REST API → `/setworkflow` ชี้ไปหา; ถ้าเป็นปุ่ม wp-admin → อ่านโค้ดปลั๊กอินเพื่อหาชื่อฟังก์ชัน แล้วเขียน snippet เรียก
- จึง **ไม่ต้อง merge โค้ดปลั๊กอิน** เข้ามา — แค่ตั้งค่าชี้ไปหามันตอนพร้อม

```
[ตอนนี้]  โครงระบบพร้อม (รีโมท+ตัวรับ) ──รอ──►  [ทีหลัง]  มีปลั๊กอิน/เว็บจริง → เสียบ → ใช้งานจริง
```
